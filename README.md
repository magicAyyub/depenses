# Application de Gestion des Dépenses Familiales

Une application web moderne développée avec **Next.js** pour gérer les dépenses familiales et éviter les disputes liées aux calculs manuels.

NB: actuellement c'est l'admin qui crée l'utilisateur à la demande, il n'y a pas d'inscription. L'emailest saisie au moment de la création du compte mais n'est pas utiliser en réalité pour garder le système simple avec juste le nom d'utilisateur et le mot de passe. Pas de mot de passe oublié non plus, il faudra contacter l'admin pour réinitialiser le mot de passe.
